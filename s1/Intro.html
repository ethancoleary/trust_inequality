Stage 1 Intro

During stage 1, you will complete as many exercises of a task for 2 minutes. The task has been selected to ensure that a good score requires solely hard effort and relies little on pre-existing skills.

You should try your best in this stage as there is a chance that your score will determine your starting position in the next stage.

Each participant will face the same exercises in the same order in this stage.

The Task

In each exercise you will need to decrypt a set of letters. You will observe a code, encryption key, and an empty text box.

The code contains 5 letters. For each letter, you need to find the corresponding decrypted letter using the encryption key and then type this letter in the empty text box.

Once you believe that you have successfully decrypted the code and typed it into the text box, you should click on 'Submit'. If your decryption is correct, then a new exercise will appear and your score will increase by 1.

How to read the key: find each letter from the code in the top row, then read the letter directly below it. Type all 5 letters together (no spaces) and click Submit.
